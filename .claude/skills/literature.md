# Literature Category Supplement

## Sectioning for Literary Authors

Organize by **individual works**:
- Each major novel/play/poem collection gets its own section
- Short works can be grouped if they share themes
- "Other Works and Collaborations" for minor pieces
- "General / Biographical" for identifiers

## Indicators

- `Work:`
- `Author:` for general/biographical
- For collaborations: `Author: co-wrote X with Y` -> just the author's name as back

## Cards

- `Novel: plot clue` -> `Work (Author)` for work-specific
- `Author: biographical clue` -> `Author Name` for general facts
- Keep full parentheticals in work names: `The Changeling (with William Rowley) (Thomas Middleton)`

## Second Pass: Expanding Work Coverage

During a second pass on literary authors, **actively try to add more work sections**:

- Scan the comprehensive summary and clue data for works mentioned but without their own section
- Query any significant work that appears in clue text but is only in "Other Works" or "General / Biographical" — if 3+ distinct clues, elevate to its own section
- For authors with large canons, run subitem queries on "Other Works" entries to check for promotion
- Lower the bar for splitting: if a work appears 2+ times across difficulties 5-10, query it and consider a dedicated section
- For poets: query individual poems that appear frequently but aren't sectioned separately

## Common Pitfalls

- For drama: distinguish between plot clues (what happens) and staging clues (how performed, notable productions)
- For poetry: individual poems vs. collections — if a poem is frequently clued on its own, give it its own section
- Classical literature uses single names (Sophocles, Aristophanes) — subcategory should be "Classical Literature"
- For movements (Confessional Poets, New Criticism): use `Movement` indicator
