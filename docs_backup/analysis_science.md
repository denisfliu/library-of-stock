# Science Analysis Guide

Supplement to `analysis_instructions.md`. Read the core instructions first.

## Sectioning for Science Topics

Science topics are usually **concepts, not creators**. Organize by:
- **Named reactions/laws/theorems** — each gets its own section if 3+ mentions
- **Applications/properties** — group related applications
- **Key experiments** — landmark experiments that are frequently clued
- **General / Giveaway Identifiers** — how the topic is defined at the end of questions

## Indicators

- `Reaction:`, `Concept:`, `Law:`, `Theorem:`, `Compound:`, `Element:`, `Organism:`
- `Scientist:` for biographical facts about people
- `Technique:`, `Process:` for methods

## Cards

- `Reaction: description` → `Reaction Name (broader topic if applicable)`
- `Concept: description` → `Concept Name`
- `Scientist: clue` → `Scientist Name`

## Filtering

Science topics are especially prone to ambiguous search results. "Alkenes" returns tossups about alkynes, pi bonds, hydrocarbons, etc. where alkenes is just a prompt/reject. **Always check the answerline** — only include questions where the topic IS the answer.

## Common Pitfalls

- Named reactions often appear in tossups for the reagent/product, not the reaction itself — cross-reference carefully
- Nobel Prize clues are high-value and often appear as power marks
- Difficulty 7-8 science is substantially different from 9-10 — the same topic may have entirely different clue sets at different levels
