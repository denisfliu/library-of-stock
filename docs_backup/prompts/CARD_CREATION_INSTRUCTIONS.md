We will now start the automated card creation process. The carding software we will be using is Anki. I primarily use two types of cards: basic and cloze.

For basic, I make my cards in the following fashion. For authors and artists and musicians, we want to be cluing their specific *works*. If it's something else, like a key or something, or anything of that nature there could be some flexibility. But for a specific work, I use the format

Front: Work: insert self contained clue 
Back: Work Name (Author/Artist/Composer) if it is art I include the image as well.

For artists, I do an extra card where the front is just a picture of the work and the back is the Work Name (Artist).

For cloze's, sometimes there is a clue with high density of key words. These are rare, but sometimes it helps to know each of these things. One example of a cloze card I have: cloze1{Martha Nussbaum} and cloze2{Amartya Sen} blah blah cloze3{capability approach} for...
